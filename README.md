# 🎬 AWS MovieFlix

MovieFlix is a movie trailer streaming website hosted on **Amazon EC2**. Movie posters and trailers are stored in **Amazon S3**, while the frontend is built using **HTML, CSS, and JavaScript**.

# 🚀 Features

- 🎥 Stream movie trailers directly from Amazon S3
- 🖼️ Display movie posters dynamically
- 📺 Popup video player
- ☁️ Amazon S3 integration
- 💻 Hosted on Amazon EC2
- 📂 File deployment using FileZilla
- 📱 Responsive user interface
